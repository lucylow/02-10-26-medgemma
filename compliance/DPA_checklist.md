# Data Processing Agreement (DPA) Checklist

**For GDPR compliance. Use with processors and subprocessors.**

---

## Items to Include

- [ ] SCCs (Standard Contractual Clauses) for international transfers
- [ ] Data transfer mechanisms (EU adequacy, SCCs)
- [ ] Subprocessors list and prior consent
- [ ] Security measures (encryption, access control)
- [ ] Data subject rights assistance
- [ ] Audit and compliance cooperation
- [ ] Breach notification (72 hours)
